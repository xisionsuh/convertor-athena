# 테스트 체크리스트

## 웹 버전 테스트 (`npm run dev`)

### 기본 기능 테스트

#### 1. 페이지 로딩
- [ ] `http://localhost:4000` 접속 시 페이지가 정상적으로 로드됨
- [ ] UI 요소들이 정상적으로 표시됨
- [ ] 사이드바가 정상적으로 표시됨

#### 2. 음성 녹음 기능
- [ ] "녹음 시작" 버튼 클릭 시 녹음 시작
- [ ] 웨이브폼이 실시간으로 표시됨
- [ ] 녹음 시간이 정상적으로 카운트됨
- [ ] "일시정지" 버튼으로 일시정지/재개 가능
- [ ] "저장" 버튼으로 녹음 저장 가능
- [ ] "저장 & 변환" 버튼으로 저장 후 자동 변환 시작
- [ ] 녹음 저장 후 세션이 사이드바에 표시됨

#### 3. 파일 업로드 기능
- [ ] 파일 선택 버튼으로 음성 파일 업로드 가능
- [ ] 여러 파일 동시 업로드 가능
- [ ] 업로드한 파일이 사이드바에 세션으로 표시됨
- [ ] 파일 크기 표시 정상

#### 4. 음성→텍스트 변환
- [ ] 세션 선택 후 "텍스트로 변환" 버튼 클릭
- [ ] 변환 진행 상태가 표시됨 (로딩 스피너, 진행 메시지)
- [ ] 변환 완료 후 텍스트가 표시됨
- [ ] 대용량 파일(25MB 초과) 자동 분할 처리
- [ ] 분할된 파일들이 순차적으로 변환됨
- [ ] 변환 중 에러 발생 시 적절한 에러 메시지 표시

#### 5. 회의록 생성
- [ ] 변환된 텍스트에서 "회의록 생성" 버튼 클릭
- [ ] 회의록 생성 진행 상태 표시
- [ ] 생성된 회의록이 구조화되어 표시됨
- [ ] 에러 발생 시 적절한 에러 메시지 표시

#### 6. 세션 관리
- [ ] 세션 선택 시 상세 정보 표시
- [ ] 여러 세션 선택 가능 (체크박스)
- [ ] 선택된 세션 일괄 변환 가능
- [ ] 세션 삭제 기능 (있는 경우)

#### 7. 데이터 저장/복원
- [ ] 페이지 새로고침 후 세션 데이터가 유지됨 (localStorage)
- [ ] 브라우저를 닫았다가 다시 열어도 세션 데이터 유지
- [ ] 손상된 데이터가 있어도 앱이 크래시하지 않음

#### 8. 에러 처리
- [ ] API 키가 없을 때 적절한 에러 메시지 표시
- [ ] 네트워크 오류 시 재시도 또는 에러 메시지 표시
- [ ] 파일 크기 초과 시 적절한 안내 메시지 표시
- [ ] 지원하지 않는 파일 형식 시 에러 메시지 표시

### UI/UX 테스트

- [ ] 반응형 디자인 (모바일, 태블릿, 데스크톱)
- [ ] 토스트 알림이 정상적으로 표시됨
- [ ] 로딩 상태가 명확하게 표시됨
- [ ] 버튼 비활성화 상태가 적절함
- [ ] 에러 메시지가 사용자 친화적임

## Electron 앱 테스트 (`npm run electron:dev`)

### 기본 기능 테스트

#### 1. 앱 실행
- [ ] 앱이 정상적으로 시작됨
- [ ] 창이 적절한 크기로 열림
- [ ] 개발자 도구가 자동으로 열리지 않음 (설정대로)
- [ ] 앱 아이콘이 표시됨

#### 2. 웹 버전과 동일한 기능 테스트
- [ ] 위의 웹 버전 테스트 항목들이 모두 동일하게 작동함
- [ ] ServiceWorker 등록 오류가 발생하지 않음

#### 3. Electron 특화 기능
- [ ] 창 크기 조절 가능
- [ ] 창 최소화/최대화 가능
- [ ] 앱 종료 시 정상적으로 종료됨

### 성능 테스트

- [ ] 앱 시작 시간이 적절함 (< 5초)
- [ ] 메모리 사용량이 적절함
- [ ] CPU 사용량이 적절함
- [ ] 대용량 파일 처리 시 앱이 멈추지 않음

## 프로덕션 빌드 테스트 (선택사항)

### 웹 빌드
```bash
npm run build
npm start
```

- [ ] 빌드가 성공적으로 완료됨
- [ ] 프로덕션 모드에서 정상 작동함
- [ ] 성능이 개발 모드와 유사함

### Electron 빌드 (macOS)
```bash
npm run electron:build
```

- [ ] 빌드가 성공적으로 완료됨
- [ ] 생성된 `.app` 파일이 실행됨
- [ ] 프로덕션 모드에서 정상 작동함
- [ ] ⚠️ macOS 26.1 (베타)에서는 크래시 가능성 있음 (알려진 문제)
